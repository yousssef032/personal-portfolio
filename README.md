This is a Vite + React + TypeScript portfolio project.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open the local URL printed by Vite (typically [http://localhost:5173](http://localhost:5173)).

You can start editing the page by modifying `src/App.tsx`. The page auto-updates as you edit files.

The project uses Google Fonts (`Inter`, `Sora`, `Geist Mono`) and Tailwind CSS v4.

## Available Scripts

- `npm run dev`: Start Vite dev server
- `npm run build`: Build for production
- `npm run preview`: Preview production build locally
- `npm run lint`: Run ESLint
